# Hardware Images

## Relay Board Photo

Place the relay board image as `relay-board.jpg` in this directory.

The image should show:
- Soldered I2C Relay Board
- I2C interface at the top
- Blue SRD-05VDC-SL-C relay module
- NO/COM/NC terminals at the bottom

**Current image:** User-provided photo of the Soldered I2C Relay board in use.
